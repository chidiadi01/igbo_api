# Examples

Base URL:
```
https://igboapi.com/api/v1/
```

***

###### Get Multiple Examples

```js
GET /examples
```
Get examples from dictionary

 **Request Headers**
| Name   | Description | Example |
| :----- | :----- | :----- |
| **X-API-Key** *required* | Secret API token | `X-API-Key=API_token` |

 **Parameters**

| Name   | Description | Example |
| :----- | :----- | :----- |
| **keyword** *required* | Keyword for querying words |  `keyword=ego` |
| **style** | Style for requested example sentences | `style=no_style` |standard|proverb|biblical` |
| **page** | Page for results | `page=1` |
| **range** | Page for result using [x,y] syntax, max range includes 25 documents | `range=[0, 24]` |


 **Responses**

 An array of up to 10 example JSON objects.

 ```js
[
  {
    "igbo": "Agụ nà-àtakwo mangō ācharọ acha",
    "english": "Agụ is eating unripe mango",
    "associatedWords": [],
    "pronunciation": "https://igbo-api.s3.us-east-2.amazonaws.com/audio-pronunciations/620c4c01ebaecf171e074c89.webm",
    "meaning": "",
    "style": "",
  },
]
 ```

 ***

 ###### Get a Single Example


```js
GET /examples/{exampleId}
```
Returns a single example JSON object from the database

**`exampleId`** is the Id of the example document you are requesting.

 **Responses**

A single example JSON object will be returned.

 ```js
{
  "nsibidi": "",
  "igbo": "Agụ nà-àtakwo mangō ācharọ acha",
  "english": "Agụ is eating unripe mango",
  "associatedWords": [],
  "pronunciation": "https://igbo-api.s3.us-east-2.amazonaws.com/audio-pronunciations/620c4c01ebaecf171e074c89.webm",
  "updatedAt": "2022-05-02T06:18:09.906Z",
  "meaning": "",
  "style": "",
  "associatedDefinitionsSchemas": [],
}
 ```
###### Proverbs
 Proverbs are gotten with the `examples` endpoint the same way examples are gotten. The only exception
 is that the `style` parameter is set to `proverb`.

 For example, this is a GET request for three (3) proverbs about `war`.

 **Parameters**

| Name   | Description | Example |
| :----- | :----- | :----- |
| **keyword** *required* | Keyword for querying words |  `keyword=war` |
| **style** | Style for requested example sentences | `style=proverb` |
| **page** | Page for results | `page=1` |
| **range** | Page for result using [x,y] syntax, max range includes 25 documents | `range=[0, 2]` |

 **Responses**
